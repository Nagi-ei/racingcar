# 기능 목록 - 자동차 경주

---

Week2 : 자동차 경주 🏎️

## 파일 구조와 역할

---

<aside>
📂 src
┣📄 App.js
┣📄 Cars.js
┗📄 Constant.js

</aside>

`App.js` : 게임 진행 (입력, 계산, 승자, 출력) 파일

`Cars.js` : 자동차 (이름, 전진 여부, 점수) 클래스 파일

`Constant.js` : 상수와 게임 진행 중 출력되는 메세지와 같은 정보가 담겨있는 파일

## 파일 별 기능 목록

---

### App.js

- [ ]  게임 시작
- [ ]  입력 (자동차 이름, 라운드)
- [ ]  점수 비교와 우승자 선정
- [ ]  라운드 별 진행 출력
- [ ]  우승자 출력

### Cars.js

- [ ]  자동차 (이름, 점수) 생성
- [ ]  자동차 라운드 진행 (전진 또는 정지)

### Constant.js

- [ ]  인게임 메세지
- [ ]  에러 메세지
